# TimeTrack

A lightweight, single-page time tracking application built purely with HTML, CSS, and JavaScript. It helps freelancers and independent professionals track the time spent on projects and automatically calculates earnings based on a customizable hourly rate.

No backend, no installation, and no database required—everything runs locally in your browser.

## Features

* **Real-time Timer:** Start, pause, and stop the timer to track active work hours.
* **Manual Entry:** Add past work sessions manually with ease.
* **Multi-Currency Support:** Choose between **USD ($)** and **BRL (R$)** as your default currency. The dashboard handles mixed-currency logs smartly.
* **Multi-Language:** Switch instantly between English and Portuguese.
* **Dashboard Overview:** Visual progress bars showing total time, earnings, and which projects are the most profitable.
* **Local Storage:** All your data is saved securely in your browser's `localStorage`. It won't disappear when you close the tab.
* **History Management:** View, edit, or delete past entries.

## How to Use

1. Clone or download this repository.
2. Open the `timetracker.html` (or whatever you named the file) directly in any modern web browser (Chrome, Edge, Firefox, Safari).
3. Start tracking your time!
